# Java Expense Tracker
Simple command-line expense tracker application built in Java using Maven.

## Features
- Add expenses (category, amount, date)
- Total expenses
- Expenses by category
- Highest & lowest spend category
- Expense trend analysis

## Tech Stack
- Java 11
- Maven
- JUnit 5

## How to Run
locate the directory with the POM file
```bash
mvn clean compile
mvn exec:java```

## How to run JUnit tests:
```mvn clean compile
mvn test```
